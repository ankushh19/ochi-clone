# Ochi Website Clone

A modern and smooth animated clone of the Ochi website built using Next.js and modern frontend technologies.

---

## 🚀 Tech Stack

✅ Next.js  
✅ React.js  
✅ TypeScript  
✅ Tailwind CSS  
✅ GSAP  
✅ Framer Motion  
✅ Locomotive Scroll  

---

## ✨ Features

- Smooth scrolling experience
- Scroll-based animations
- Responsive UI
- Modern landing page design
- Interactive transitions
- Performance optimized with Next.js

---

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/ankushh19/ochi-clone.git

npm i

npm run dev
```
